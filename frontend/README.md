# RAG Employee Search - Frontend

Frontend for the RAG-powered employee search application.

## Deploy to Vercel

1. **Update API URL**: Edit `index.html` line 96 and replace `YOUR_RENDER_APP_NAME` with your actual Render app name

2. **Deploy**:
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import this repository
   - Set root directory to `frontend`
   - Deploy!

## Local Development

Simply open `index.html` in a browser (make sure to update the API URL first).
